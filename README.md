# Formulário-de-Pesquisa-Programação

Este é um projeto de Formulário de Pesquisa desenvolvido como parte da certificação de Responsive Web Design do freeCodeCamp. O objetivo foi criar um formulário funcional, acessível e semanticamente correto utilizando HTML5 e CSS3.

Tecnologias Utilizadas
HTML5: Estruturação semântica de dados.
Acessibilidade: Uso correto de labels, ids e atributos de validação.
